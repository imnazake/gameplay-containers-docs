Detailed Description

UEquipmentComponent

A class that manages equipment within a gameplay container, providing functionalities for equipment registration, equipping, and unequipping items.

Note
This requires an inventory component to be registered for it work properly.
An item is considered equipped as soon as its placed in one of this container slots and the item equipment slot matches the slot type its in.
Unequipping any item from any slot within this container will result in forced returning of the item to the registered inventory component.
Use each slot definition to customize the filter to accept only the items you want the player to be able to equip in here
If there's no inventory component registered the items will be just removed from this container, you can perform saving before desroying items
