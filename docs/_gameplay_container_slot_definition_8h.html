EGameplayContainerSlotFilteringPolicy

An enumeration defining different policies for filtering items in a gameplay container slot. These policies determine what types of items can be stored in a particular slot.

Enumerator
AcceptOnlySpecificItems 

Only the specified items in the Items array can be stored in this slot.

RejectOnlySpecificItems 

Only the specified items in the Items array cannot be stored in this slot.

AcceptEverything 

Anything can be stored in this slot.

RejectEverything 

Nothing can be stored in this slot.
