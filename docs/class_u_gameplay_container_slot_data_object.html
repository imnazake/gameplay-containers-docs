Detailed Description

UGameplayContainerSlotDataObject

Data object representing a gameplay container slot.

This class is used to pass information about a gameplay container slot between different parts of the game, such as during drag-and-drop operations or UI updates. It provides access to the slot's details, including its definition, item, and owner container.
