Detailed Description

UGameplayItemFragment_UI

Represents a user-interface fragment associated with a specific gameplay item, determining how the item is displayed on the user interface. This class can be utilized to define the visual representation and behavior of an item when displayed in the user interface. Developers can use this class to customize and control the appearance and behavior of gameplay items when presented to the user.
